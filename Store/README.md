This is a Full-Stack E-commerce application developed as part of the Web Development course. It features a seamless integration between a **Django REST Framework** backend and an **Angular** frontend.

* **Authentication:** JWT-based login/logout system.
* **Product Management:** Full CRUD operations via Django Admin.
* **Shopping Cart:** Real-time cart management with coupon and note features.
* **Dynamic UI:** Responsive design with modern CSS (Glassmorphism and Gradients).

- **Angular:** 4+ API events, 4+ Form inputs (`ngModel`), Routing, Services.
- **Django:** 4+ Models, ForeignKey relations, 2+2 Serializers, CBVs & FBVs.
- **Tools:** Postman collection included, CORS enabled.
